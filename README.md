# bot-discord

Un bot Discord minimal en JavaScript (discord.js).

## Prérequis

- Node.js 18+ installé
- Un token de bot Discord (Portail des développeurs Discord)

## Configuration

1. Créez un fichier `.env` à la racine avec:

   ```
   DISCORD_TOKEN=your-bot-token
   ```

2. Invitez le bot à votre serveur avec les permissions de base (lecture/écriture de messages).

## Installation

```powershell
Push-Location "c:\Users\aland\Desktop\Nouveau dossier (3)"
npm install
```

## Démarrage

```powershell
npm run start
```

Le bot répond à `!ping` et `!echo <texte>`.

## Développement (auto-reload)

```powershell
npm run dev
```
